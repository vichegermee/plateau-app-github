grunt initConfig

Description

A linterieur de la fonction wrapper on a un bloc (initConfig) et un ensemble de tache personnalisée. Ce bloc (initConfig) permet de configurer l'ensemble des tâches lancées par Grunt.

Example

// injection de dependances dans le fichier index.html
wiredep : {
app : {
src : [ '<%= yeoman.app %>/index.html' ],
ignorePath : /\.\.\//
},
